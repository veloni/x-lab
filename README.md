# О проекте

1) Приложение создано при помощи `React Hook`, хуки передаются через компоненту `Body` из одной компоненты в другую.

2) После захода сайта, появляется форма логина и пароля, в случае неправильности ввода, приходит сообщение о неправильности данных

3) В случае коректо введеных данных происходит переход к списку пользователя, и запись `key`, в `localstorage`, после повторного захода, если `key` существует, переход к списку происходит сразу

4) Сортировка `Id` происходит по стрелке, фильтрация по `userName` в `input`, если данные введеные в `input` содержатся в `userName`, то пользователи отображаются

5) Создавал с помощью `create-react-app`, запуск  `npm start`
